# 📄 AI Resume Analyzer

An AI-powered resume analyzer that provides instant feedback 
to help job seekers optimize their resumes.

## 🚀 Features
- Upload resume as PDF
- AI-powered analysis using Groq (LLaMA 3)
- Overall score out of 10
- Skills extraction
- Strengths & weaknesses
- ATS friendliness score
- Improvement suggestions

## 🛠️ Tech Stack
- Python
- Streamlit
- Groq API (LLaMA 3.3)
- PyMuPDF

## ⚙️ Setup
1. Clone the repo
2. Install dependencies: `pip install -r requirements.txt`
3. Create `.env` file and add: `GROQ_API_KEY=your-key-here`
4. Run: `streamlit run main.py`
